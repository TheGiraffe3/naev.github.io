Detailed Description

The state of a line iteration. This matches the process of rendering text into an on-screen box: An empty string produces a zero-width line. Each regular, fitting character expands its line horizontally. A newline or wrapping leads to vertical expansion. The layout calculation is iterative; one may for instance change the width limit between lines.

gl_printLineIteratorInit, gl_printLineIteratorNext.
