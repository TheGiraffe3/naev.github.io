Renders a texture.

This function has variable parameters depending on how you want to render.

Lua function parameter: Tex tex Texture to render. Lua function parameter: number pos_x X position to render texture at. Lua function parameter: number pos_y Y position to render texture at. Lua function parameter:[opt=0] int sprite_x X sprite to render. Lua function parameter:[opt=0] int sprite_y Y sprite to render. ◆ gfxL_renderTexRaw()

static int gfxL_renderTexRaw ( lua_State *  L)
static

Renders a texture using the core render function.

This function is far more complex than renderTex, however it allows much more fine grained control over the entire render process and puts you closer to the actual OpenGL calls.

Lua usage parameter: gfx.renderTexRaw( tex, 0., 0., 100., 100., 1, 1, 0., 0., 0.5, 0.5 ) – Renders the bottom quarter of the sprite 1,1 of the image.

Lua function parameter: Tex tex Texture to render. Lua function parameter: number pos_x X position to render texture at. Lua function parameter: number pos_y Y position to render texture at. Lua function parameter: number pos_w Width of the image on screen. Lua function parameter: number pos_h Height of the image on screen. Lua function parameter: number sprite_x X sprite to render. Lua function parameter: number sprite_y Y sprite to render. Lua function parameter: number tex_x X sprite texture offset as [0.:1.]. Lua function parameter: number tex_y Y sprite texture offset as [0.:1.]. Lua function parameter: number tex_w Sprite width to display as [-1.:1.]. Note if negative, it will flip the image horizontally. Lua function parameter: number tex_h Sprite height to display as [-1.:1.] Note if negative, it will flip the image vertically. Lua function parameter:[opt] Colour colour Colour to use when rendering. Lua function parameter:[opt] number angle Angle to rotate in radians.
