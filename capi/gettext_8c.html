void gettext_init ()
 Initialize the translation system. There's no presumption that PhysicsFS is available, so this doesn't actually load translations. There is no gettext_exit() because invalidating pointers to translated strings would be too dangerous. We loosely follow: https://www.gnu.org/software/gettext/manual/html_node/Locale-Environment-Variables.html.
